This is a [Giter8][g8] template for Java

```
sbt new sbt/java-seed.g8
```

There are templates for sbt 1.x and 0.13.x.

* 1.x
    ```bash
    sbt new sbt/java-seed.g8 -b sbt1
    ```
* 0.13.x
    ```bash
    sbt new sbt/java-seed.g8 --branch sbt0.13
    ```
